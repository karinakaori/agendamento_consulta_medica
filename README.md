# Agendamento de Consulta Medica

## Descricao do projeto

Este repositorio apresenta os artefatos da atividade pratica da Unidade III, aplicada ao gerenciamento de um projeto de software com apoio de inteligencia artificial generativa.

O cenario analisado e o desenvolvimento de um aplicativo movel para agendamento de consultas medicas. O sistema contempla cadastro de usuarios, gestao de agenda medica, agendamento de consultas, notificacoes e integracao com um sistema externo de prontuario.

O projeto esta em fase intermediaria de desenvolvimento. Parte das funcionalidades ja foi implementada, mas existem pontos criticos relacionados a integracao externa, mudancas de requisitos e aumento da carga de trabalho da equipe.

## Objetivo do exercicio

O objetivo e usar um LLM como apoio para:

- identificar riscos do projeto;
- analisar qualitativamente impactos e fatores condicionantes;
- definir estrategias de resposta aos riscos mais relevantes;
- elaborar uma comunicacao objetiva para stakeholders;
- organizar a documentacao em Markdown, mantendo rastreabilidade dos artefatos produzidos.

## Organizacao do repositorio

- `riscos/identificacao.md`: lista dos riscos identificados, com descricao e contexto.
- `riscos/analise.md`: analise qualitativa dos riscos, impactos e fatores condicionantes.
- `riscos/respostas.md`: estrategias de resposta, justificativas e acoes associadas.
- `comunicacao/status_stakeholders.md`: comunicado objetivo para stakeholders.
- `prompts/prompts_utilizados.md`: exemplos de prompts usados para apoiar a producao dos artefatos.

## Observacao sobre uso de IA

A IA generativa foi utilizada como apoio para estruturar ideias, organizar a analise e redigir os artefatos. As decisoes finais foram revisadas considerando o contexto do projeto, a criticidade da integracao com prontuario e a capacidade da equipe descrita no cenario.
